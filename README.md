# ✈️ Boing Boing Airlines - Management System

A comprehensive, full-stack airline management and booking system featuring a modern, obsidian-themed operational dashboard and seamless ticket generation. 

This application allows users to browse flights, book reservations, generate detailed boarding passes, and provides administrators with a powerful ground control dashboard to monitor live departures, fleet statistics, and announcements.

## ✨ Key Features

* **Real-time Operations Dashboard:** Monitor total flights, domestic/international splits, on-time percentages, and live departure statuses.
* **Flight Search & Booking Engine:** Users can search for specific routes, select cabin classes, and book tickets seamlessly.
* **Dynamic Boarding Passes:** Automated generation of detailed flight stubs including seat assignments, gate numbers, and PNR codes.
* **Customer Management:** Securely add, update, and manage passenger details.
* **Responsive UI/UX:** Clean, modular, and professional design favoring an obsidian-themed interface for optimal user experience.

## 🛠️ Tech Stack

**Frontend:**
* React.js (Vite)
* HTML5 / CSS3 (Responsive Design)
* Axios (API calls)
* Deployed on **Vercel**

**Backend:**
* Node.js
* Express.js
* RESTful API Architecture
* Deployed on **Render**

**Database:**
* MySQL (Hosted on **Aiven**)
* Seamless cloud synchronization
